# Saree4ever

E-commerce platform for sarees built with Next.js and Node.js/Express.

## Project Structure

```
saree4ever/
├── frontend/     # Next.js app (Vercel)
├── backend/      # Node/Express API (Render)
└── docs/         # Documentation & notes
```

## Tech Stack

### Frontend
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

### Backend
- **Runtime:** Node.js
- **Framework:** Express
- **Language:** TypeScript
- **Deployment:** Render

## Getting Started

### Frontend Development

```bash
cd frontend
npm install
npm run dev
```

Visit http://localhost:3000

### Backend Development

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

API runs on http://localhost:5000

## Deployment

- **Frontend:** Deploy to Vercel (automatic from GitHub)
- **Backend:** Deploy to Render (connect GitHub repo)

See individual README files in each directory for detailed setup instructions.
