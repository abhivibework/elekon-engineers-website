# Quick Start: Supabase MCP Server

## Fast Setup (5 minutes)

### Step 1: Open Cursor Settings
- Press `Cmd + ,` (Mac) or `Ctrl + ,` (Windows/Linux)

### Step 2: Add MCP Server

**Option A: One-Click (Easiest)**
1. Visit [Supabase MCP Docs](https://supabase.com/docs/guides/getting-started/mcp)
2. Select Cursor as your client
3. Click **"Add to Cursor"** button

**Option B: Manual Setup**
1. Go to `Features` â†’ `MCP`
2. Click "Add new MCP Server"
3. Configure:
   - **Name**: `supabase`
   - **Type**: `http`
   - **URL**: `https://mcp.supabase.com/mcp`

### Step 3: Authenticate
- Cursor will open a browser window
- Log in to your Supabase account
- Grant access to the MCP client
- Select your organization

### Step 4: Verify
Ask the AI assistant:
- "List my Supabase projects"
- "Show my database tables"
- "Query my Supabase database"
